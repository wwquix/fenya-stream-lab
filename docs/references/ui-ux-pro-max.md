# UI/UX Pro Max Reference

Use this file as design guidance only.
Do not copy it as code.

## Visual direction

The site should feel like a premium streamer analytics product, not a generic gaming landing page.

Target feeling:
- Apple-like clarity;
- fintech dashboard;
- Hyperliquid-style dark analytics;
- premium creator tool;
- clean, calm, expensive.

## Design rules

Use:
- dark glass cards;
- subtle borders;
- soft shadows;
- calm spacing;
- clear hierarchy;
- large readable section titles;
- compact but readable data labels;
- smooth hover states;
- subtle lime/cyan/purple accents.

Avoid:
- cheap neon;
- excessive glow;
- random white stripes;
- huge noisy gradients;
- overcomplicated backgrounds;
- too many animations at once;
- admin-panel look;
- tiny unreadable labels;
- broken-looking decorative pseudo-elements.

## Cards

Cards should:
- have enough padding;
- have clear title/value/subtitle hierarchy;
- use small badges only when useful;
- not overflow;
- not break short Russian labels;
- feel like part of one system.

## Data UI

Charts and stats should explain themselves.

Bad:
- “Medium”
- “Stable”
- “Minecraft”
without context.

Better:
- “Нагрузка модеров: средняя”
- “Состояние стрима: активно, стабильно”
- “Главная категория: Minecraft”

## Motion

Motion should support usability, not distract.

Good:
- fade in;
- slide up;
- subtle hover lift;
- small arrow movement;
- smooth dropdown expansion.

Bad:
- huge bouncy animations;
- strong scaling;
- constant moving background;
- motion that hides data.
