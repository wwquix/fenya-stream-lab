# Assets

This folder is reserved for future Fenya Stream Lab visual assets.

Current MVP rule: the hero supports a future real image or video asset, but the active background is CSS-only. Do not add heavy particle systems, Three.js, Spline, Twitch, YouTube, or Kick assets in this first version.

## Hero Asset Slot

Place the final `hero-background.webp` or `hero-background.mp4` in this folder, then update `src/components/Hero.jsx` at the `defaultHeroAsset` swap point.

Recommended asset direction:

- calm premium streamer analytics scene;
- soft gray, graphite, blue, and restrained green tones;
- wide cinematic composition that works behind centered typography;
- enough empty space for glass cards and navigation;
- optimized file size before commit.

Planned asset candidates:

- abstract dashboard background stills;
- subtle streamer analytics video loop;
- premium chart and archive thumbnails;
- future liquid surface reference renders.
